# INFP

他们没有特别喜欢的东西，没有非做不可的事情，不喜欢孤独，但享受孤独。

表面上乐观共情，实际上悲观淡漠，，凡事都做最坏的准备。

调停型人格，把内在的和谐，视为高于其他一切，他们敏感，理想化，忠诚，对于个人价值，具有强烈的荣誉感。

是蝴蝶作茧自缚，背负着自己给自己的枷锁。
