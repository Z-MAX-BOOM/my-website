# 睡眠安排

## [ 睡眠计算器](https://sleepcalculator.com/)

1. 如何设定一个合理的睡眠，关键在于掌握**睡眠周期**，睡眠周期是指睡眠过程中**浅睡眠、深睡眠、快速眼动睡眠（REM）** 三个阶段循环往复的节律，一个完整的睡眠周期时长约 **90 分钟**，成年人夜间睡眠通常会经历 **4–6 个** 这样的周期（6小时-9小时）。

2. 睡眠周期的夜间变化规律

   - 入睡后先进入**浅睡眠→深睡眠→浅睡眠**，再进入第一次 REM 睡眠，完成一个 90 分钟的周期。
   - 随着夜间时间推移，**深睡眠的占比会逐渐减少，REM 睡眠的占比会逐渐增加**；前半夜以深睡眠为主，后半夜以 REM 睡眠为主。
   - 这也是为什么熬夜后，补觉更推荐睡够 90 分钟的完整周期 —— 能同时兼顾深睡眠的体力修复和 REM 睡眠的记忆巩固。

3. 保证与睡眠周期匹配

   - 固定入睡和起床时间，让身体形成稳定的周期节律，比如每天 23 点睡、7 点起（约 5 个睡眠周期）。

   - **午睡**尽量不超过**30 分钟**，避免打断深睡眠；**熬夜后补觉**可选择 **90 分钟**的完整周期，减少睡眠惯性。

   - 起床时间尽量选在睡眠周期的结束点（比如入睡后 90 分钟、180 分钟），醒后会更清醒。
